# Sleep2Wake

> 온라인 수업 중 학생의 집중 저하, 졸음, 자리 이탈을 실시간으로 감지하고  
> 강사와 관리자가 즉시 대응할 수 있도록 돕는 AI 기반 학습 모니터링 플랫폼

## 1. 프로젝트 소개

**Sleep2Wake**는 비대면 수업 환경에서 발생하는 학습 몰입도 저하 문제를 해결하기 위해 만든 서비스입니다.  
웹캠 기반 얼굴 랜드마크 분석을 통해 학생의 상태를 실시간으로 판정하고, 그 결과를 강사 화면과 관리자 대시보드에 반영합니다.

단순히 “졸음 여부”만 보는 것이 아니라, 실제 수업 상황에서 자주 발생하는 **필기, 잠깐 아래 보기, 자리 비움, 시선 이탈, 눈 감김** 같은 맥락까지 고려해 보다 자연스러운 상태 판정을 목표로 했습니다.

### 핵심 목표
- 학생의 집중 상태를 **실시간으로 시각화**
- 강사가 수업 중 즉시 개입할 수 있도록 **라이브 모니터링 제공**
- 수업 종료 후 세션 단위 데이터를 축적해 **관리자 관점의 분석 지원**
- 브라우저 기반으로 동작하여 **별도 설치 없이 접근 가능**

---

## 2. 주요 기능

### 학생 화면
- 웹캠을 통한 얼굴 인식 및 상태 판정
- 집중 / 시선 이탈 / 졸음 의심 / 졸음 확정 / 얼굴 미감지 / 자리 이탈 표시
- 눈 감김, 자리 이탈 누적 타이머 제공
- 강사 자막 실시간 표시 및 자막 On/Off 토글
- 강사 음성 및 참여자 음성 수신
- 마이크/카메라 On/Off
- 스트레칭 / 쉬는 시간 오버레이 수신
- 보조 학생 화면 페이징 기능

### 강사 화면
- 학생별 상태 실시간 모니터링
- 학생 비디오 피드 확인
- 수업 코드 생성 및 입장 관리
- 마이크/카메라/화면 공유 제어
- 자막 송출 및 자막 표시 On/Off
- 스트레칭 / 쉬는 시간 수동 전송
- 수업 종료 후 관리자 대시보드로 이동

### 관리자 화면
- 진행 중인 세션 확인
- 실시간 학생 상태 모니터링
- 세션/과정별 조회
- 집중도 및 상태 변화 기반 데이터 확인
- 수업 종료 후 누적 데이터 기반 분석 지원

## 3. 상태 판정 로직

Sleep2Wake는 단순한 1프레임 판정이 아니라, **시간 누적 기반 상태 머신**으로 동작합니다.

### 대표 상태
- `집중`
- `시선 이탈`
- `상태 판독 중`
- `졸음 의심`
- `졸음 확정`
- `얼굴 미감지`
- `자리 이탈`

### 현재 주요 기준
- 눈 감김 지속 `5초` 이상: `졸음 의심`
- 눈 감김 지속 `10초` 이상: `졸음 확정`
- 얼굴 미감지 `5초` 이상: `자리 이탈`
- 짧은 눈 감김/필기 자세 등 애매한 구간: `상태 판독 중`

### 오탐 방지 아이디어
- 필기 중 고개를 숙이는 상황을 고려한 EAR threshold 보정
- 필기와 졸음을 구분하기 위한 `상태 판독 중` 단계 도입
- 턱을 괴는 자세는 시선 이탈로 과도하게 잡히지 않도록 완화
- 얼굴 미감지 직후 바로 자리 이탈로 가지 않고 유예 시간 적용
